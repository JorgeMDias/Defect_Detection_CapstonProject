Notebooks used for Multiclassification Models
You will be able to find 4 notebooks, once for each method attempted to deal with class imbalance
The methods attempted to were:
Class Weights
Bootstrapping with Class Weights
Undersampling with GridSearchCV and RFECV
Oversampling (SMOTE)

Each notebook contains models for each of the following algorithms:
Random Forest Classifier
KNN
XGBoost
Extra Trees Classifier
Voting Classifier
